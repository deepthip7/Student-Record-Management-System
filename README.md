# Student-Record-Management-System

-> Project Description

The Student Record Management System is a Java-based console application designed to store, manage, and retrieve student information efficiently. This system helps in maintaining student records such as name, roll number, course, and marks using appropriate data structures and file handling (if implemented). It provides an easy-to-use interface for performing basic CRUD operations.

->Objectives

To digitally manage student records

To understand Object-Oriented Programming (OOP) in Java

To perform Create, Read, Update, and Delete (CRUD) operations

To improve data handling efficiency

To reduce manual record maintenance

-> Technologies Used

Programming Language: Java(Backend), html,css,javascript(frontend)

IDE: IntelliJ IDEA

Platform: Console-based Application

-> Features

Add new student record
 
View all student records
 
Update student details

Delete student record
 
Search student by Roll Number
